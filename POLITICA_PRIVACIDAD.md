# Poloticas de privacidad TripCounter-V2 #
Esta política es específica para una aplicación de backend que utiliza una Cuenta de Servicio.
​[Nombre de la Aplicación]: TripCounterapp
https://ajmonteroa45-del.github.io/tripcounterapp/
[Fecha de Vigencia]: 2 de noviembre de 2025
​Resumen de Recopilación de Datos
​Esta Política de Privacidad describe cómo la aplicación TripCounter-V2 (en adelante, "la Aplicación") recopila, utiliza y protege la información al interactuar con las API de Google, específicamente Google Sheets y Google Drive.
​1. Información que Recopilamos
​La Aplicación es un sistema de automatización de backend que utiliza una Cuenta de Servicio para procesar datos. Por lo tanto, no recopila directamente información personal identificable (IPI) de usuarios finales (nombres, correos electrónicos, contraseñas, etc.).
​La Aplicación accede únicamente a los siguientes datos a través de la Cuenta de Servicio:
​Datos de Google Sheets: Contenido de celdas, estructuras de hojas y metadatos (nombres y formatos de las hojas) de las hojas de cálculo compartidas directamente con la Cuenta de Servicio.
​Datos de Google Drive: Permisos para crear, copiar, mover y eliminar archivos de hojas de cálculo según sea necesario para el flujo de trabajo automatizado.
​2. Cómo Usamos la Información
​Los datos accedidos (Hojas de Cálculo y Drive) se utilizan únicamente para el propósito declarado de la Aplicación:
​Sincronización de Datos: Para leer la configuración y las plantillas.
​Automatización: Para insertar nuevos registros, actualizar datos existentes y organizar archivos generados (informes) en carpetas de Drive específicas.
​3. Compartir y Almacenar Información
​Compartir: La Aplicación no comparte los datos de Google Sheets o Drive con terceros. Los datos se mantienen estrictamente dentro del ecosistema del proyecto.
​Almacenamiento: Los datos se procesan en tiempo real y no se almacenan fuera del proyecto de Google Cloud o de las Hojas de Cálculo de Google.
​4. Cumplimiento de las Políticas de Uso de Datos de Google
​La Aplicación se adhiere a la Política de Datos de Usuario de Google y a las directrices de la API de Google Sheets y Drive. Los ámbitos sensibles (/spreadsheets y /drive) se utilizan solo para proporcionar la funcionalidad principal del backend, tal como se describe en esta política.
